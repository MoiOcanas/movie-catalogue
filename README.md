# Movie Catalogue App

Movie app using the themoviedb API, React, Redux, React hooks and SASS.

## Installation

Clone the repository:

```bash
git clone https://github.com/MoiOcanas/movie-catalogue.git
```

Run: 

```bash
npm install
```

And finally run: 

```bash
npm start
```
